##  Features ✨

- 🛍️ Book browsing by categories
- 🔍 Search functionality
- 🛒 Shopping cart system
- 💳 Checkout process
- 📱 Mobile-friendly design
- 🌙 Dark/Light mode toggle
- ⭐ Book rating system

##  Technologies Used 🛠️

- **Frontend**: HTML5, CSS3, JavaScript
- **UI Framework**: Bootstrap 5
- **Icons**: Font Awesome
- **Hosting**: GitHub Pages
